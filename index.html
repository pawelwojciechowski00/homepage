HTLM

Co to jest HTML?

HTML - HyperText Markup Language
Jest semantyczny = mający znaczenie.
Odpowiada za treść (a właściwie za jej opakowanie). Treść do teksty, nagłówki, linki, listy, definicje, obrazki, video, formularze.
Ogólnie wszystko co jest nośnikiem informacji.
HTML składa się ze znaczników/tagów/elementów. Pliki zawierające kod HTML mają zwykle rozszerzenie .html
Więcej na ten temat dowiesz się tutaj
Poczytaj teraz o css

Elementy HTML

main - główna część dokumentu
section - sekcje dokumentu
article - samodzielne treści
header - nagłówek dokumentu/sekcji/artykułu
footer - stopka dokumentu/sekcji/artykułu
h1-h6 - tekstowe nagłówki tytułowe dokumentu/sekcji/artykułu
p - paragraf/akapit

CSS

CSS - Cascading Style Sheets (Kaskadowe arkusze stylów)
CSS odpowiada za prezentację, wpływają na wygląd elementów HTML na ekranie, wydrukach itp.
Składnia to reguły np. selektor{właściwość: wartość;}. Pliki mają rozszerzenie .css.
